# Acquired genetic and cell state changes in IDH-mutant glioma progression

## Overview
The Cellular Analysis of Resistance and Evolution (CARE) consortium used bulk and single-nucleus sequencing approaches to study IDH-mutant glioma evolution. This repository contains scripts for analyzing single-nucleus RNA and ATAC sequencing data from IDH-mutant gliomas and experimental models. Pre-processing analyses related to bulk DNA sequencing can be found [here](https://github.com/Kcjohnson/care-glass).

## Data availability
Raw data (fastqs/bams) for DNA and single-nucleus RNA/ATAC sequencing data can be accessed via the [European-Genome Phenome Archive](https://ega-archive.org/studies/EGAS50000001727) and [DUOS](https://app.terra.bio/#workspaces) (DUOS-000475 and DUOS-000477).

The processed gene expression count matrices and ATAC fragment files can be found on the Gene Expression Omnibus under the following accessions: [GSE326221](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE326221) and [GSE327580](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE327580)
